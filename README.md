# Installation:
```
git clone https://github.com/hiddify/hiddify_support_bot

cd hiddify_support_bot
cp .env_example .env
#Enter the user panel link and admin panel link and bot token in .env file
nano .env 
pip install -e .
hiddify_support_bot
```
